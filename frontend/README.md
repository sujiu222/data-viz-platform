# 前端可视化应用

## 启动说明

1. 安装依赖：

```bash
npm install
```

2. 启动开发服务器：

```bash
npm run dev
```

3. 构建生产版本：

```bash
npm run build
```

4. 预览构建结果：

```bash
npm run preview
```

## 端口

开发服务器端口：3000

## 技术特性

- Vue 3 Composition API
- Vite 快速构建
- ECharts 数据可视化
- Element Plus UI 组件
- Vue Router 路由管理
- Axios HTTP 请求
- SCSS 样式预处理
